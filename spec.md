# FFmpeg Web UI 音视频处理工具 - 需求规范

## 一、项目概述

### 1.1 角色定位
全栈开发工程师，精通Python FastAPI后端、Vue3前端、FFmpeg音视频处理、异步任务系统设计、网络安全防护。

### 1.2 项目背景
FFmpeg是音视频处理领域的事实标准，但命令行操作门槛极高，普通用户无法便捷使用。

### 1.3 核心目标
为FFmpeg命令行工具开发一套易用的Web可视化界面，让用户通过浏览器点击操作，即可完成常用音视频处理任务，无需记忆复杂的FFmpeg命令。项目必须开箱即用，最小化用户部署成本，同时保证安全性、稳定性、易用性。

## 二、核心功能模块需求

### 模块1：系统初始化与环境检测模块

1. **环境检测**
   - 服务启动时自动检测系统FFmpeg安装状态
   - 未安装时在控制台和前端系统页面给出全平台（Windows/Linux/macOS）明确的安装指引

2. **自动初始化**
   - 自动创建项目所需的storage目录及子目录
   - 自动初始化数据库，无需用户手动操作

3. **系统配置**
   - 提供系统配置页面
   - 可可视化修改文件大小上限、并发任务数、FFmpeg路径等核心配置

4. **用户认证**
   - 实现基础的用户注册/登录功能
   - 基于JWT鉴权
   - 实现用户数据完全隔离

### 模块2：用户文件管理模块

1. **上传功能**
   - 支持大文件拖拽上传
   - 支持分片上传、断点续传
   - 默认最大支持2GB文件上传，上限可配置

2. **文件校验**
   - 上传时自动校验文件头
   - 仅允许音视频、图片类合法文件
   - 非法文件直接拦截并给出明确提示

3. **元数据解析**
   - 上传完成后自动调用ffprobe解析文件元数据
   - 展示文件名、大小、时长、分辨率、编码格式、码率、帧率、声道数等核心信息

4. **文件操作**
   - 支持音视频/图片文件在线预览
   - 支持文件重命名、删除
   - 支持批量管理、搜索筛选

5. **权限控制**
   - 所有文件操作必须做鉴权校验
   - 用户只能操作自己的文件，禁止越权访问

### 模块3：视频处理核心模块

#### 3.1 视频剪切/拆分/合并功能

1. **剪切功能**
   - 可视化时间轴选择开始/结束时间
   - 支持精确到毫秒的时间码输入
   - 支持实时预览截取片段

2. **剪切模式**
   - 无损剪切模式：FFmpeg流复制（-c copy），极速完成无画质损失，仅支持关键帧剪切
   - 重新编码剪切模式：精确到帧剪切，可自定义编码参数，画质可控

3. **拆分/合并**
   - 支持大视频按时长/文件大小批量拆分
   - 支持多视频无缝合并（无损/重新编码双模式）

#### 3.2 视频格式/编码转换功能

1. **封装格式**
   - 支持MP4、MKV、AVI、MOV、FLV、TS、WebM、WMV等主流格式

2. **视频编码**
   - 支持H.264/AVC、H.265/HEVC、H.266/VVC、AV1、VP9、MPEG-4等主流编码

3. **音频编码**
   - 支持AAC、MP3、Opus、FLAC、WAV、AC3等主流编码

4. **一键预设**
   - 提供常用一键预设：转MP4(H264+AAC)、HEVC压缩、转WebM、转GIF动图等

5. **自定义参数**
   - 支持自定义核心参数：分辨率、帧率、码率、CRF值、GOP结构、音频采样率、声道数等

6. **轨道管理**
   - 支持音轨、字幕轨道的选择、保留、删除
   - 支持音画同步调整

#### 3.3 视频画面调整功能

1. **分辨率/比例**
   - 支持视频分辨率缩放
   - 支持画面比例调整
   - 支持等比例缩放和强制拉伸

2. **裁剪**
   - 支持画面裁剪
   - 可视化选择裁剪区域
   - 去除黑边/多余画面

3. **变换**
   - 支持画面旋转、水平/垂直翻转
   - 支持帧率调整、补帧/降帧
   - 支持慢动作/快放速度调整

#### 3.4 视频滤镜与水印功能

1. **文字水印**
   - 自定义内容、字体、大小、颜色、透明度、位置、显示时间段

2. **图片水印**
   - 自定义水印图片、位置、透明度、缩放比例、显示时间段

3. **基础滤镜**
   - 画面调色（亮度/对比度/饱和度/色温）
   - 锐化、模糊、去噪、去隔行、HDR转SDR

### 模块4：音频处理核心模块

1. **音频提取**
   - 从视频中无损提取音频流
   - 支持输出MP3、AAC、FLAC、WAV、Opus等格式

2. **音频剪切/合并**
   - 截取音频片段
   - 多段音频无缝合并

3. **音频参数调整**
   - 音量调节、淡入淡出
   - 声道格式转换、采样率调整

4. **音频特效**
   - 变速不变调、变调不变速
   - 降噪、去人声、均衡器调节

5. **音轨替换**
   - 给视频替换背景音乐
   - 支持音量配比调整

### 模块5：图文与视频互转模块

1. **视频转图片**
   - 按时间间隔、指定时间点、按帧提取视频帧
   - 输出JPG/PNG格式
   - 支持批量打包下载

2. **图片转视频**
   - 多张图片按顺序合成视频
   - 支持设置帧率、转场效果、添加背景音乐

3. **GIF制作**
   - 视频片段转高质量GIF
   - 支持自定义帧率、分辨率、播放次数

### 模块6：任务中心与进度管理模块

1. **异步任务**
   - 所有音视频处理任务全部通过Celery异步执行
   - 禁止阻塞Web服务

2. **任务列表**
   - 展示用户所有任务
   - 包含任务类型、源文件、创建时间、状态（排队中/处理中/已完成/失败）、处理进度

3. **实时进度**
   - 通过WebSocket实时向前端推送任务进度
   - 进度条精确到1%
   - 实时展示FFmpeg执行日志

4. **任务管理**
   - 支持暂停/继续（断点续传）
   - 支持终止、删除、重试失败任务

5. **任务完成**
   - 支持输出文件在线预览
   - 一键下载、重新处理

6. **权限控制**
   - 所有任务操作必须做鉴权校验
   - 用户只能操作自己的任务

## 三、核心技术实现要求

### 3.1 FFmpeg命令封装
- 所有FFmpeg功能必须封装为独立Python函数
- 单函数单职责
- 输入参数经过白名单校验
- 输出标准化的FFmpeg命令列表

### 3.2 进度解析
- 必须实现FFmpeg输出精准解析
- 正确解析time=字段
- 结合文件总时长计算处理进度
- 进度计算误差≤1%

### 3.3 WebSocket推送
- 必须实现任务ID与用户绑定
- 每个用户只能收到自己的任务进度
- 禁止串线

### 3.4 大文件处理
- 必须支持分片上传、断点续传
- 大文件处理时不会占用过高内存

### 3.5 性能优化
- FFmpeg命令必须做最优配置
- 无损剪切使用-ss在前-i在后的参数顺序，提升处理速度
- 自动检测并适配主流硬件加速编解码方案

## 四、验收标准

1. 执行一键启动脚本，后端、Celery Worker、前端能正常启动无报错，控制台无异常
2. 系统能自动检测FFmpeg安装状态，未安装时给出明确的全平台安装指引
3. 能正常上传最大2GB的音视频文件，支持拖拽上传，上传后能正常解析元数据、在线预览
4. 视频剪切功能：无损剪切输出文件无画质损失、音画同步、时长准确；重新编码剪切精确到帧
5. 格式转换功能：转换后的文件能正常播放，参数完全符合设置要求
6. 所有耗时任务都能在任务中心查看实时进度，进度条实时更新无延迟，能查看完整执行日志
7. 任务完成后，能正常在线预览、下载输出文件，文件无损坏、无缺失
8. 异常场景处理：上传非法文件、输入非法参数、FFmpeg执行失败，都有明确错误提示，不会导致系统崩溃、服务宕机
9. 安全合规：所有FFmpeg命令均为列表形式参数化调用，无命令注入漏洞；用户输入全量白名单校验；用户文件和任务完全隔离，无越权访问漏洞
